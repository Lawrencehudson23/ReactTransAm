# ReactTransAm
A clone of the Spectrum game TransAm written in ReactJS


# To Run

1. Clone the code to a local repo

2. If you don't already have npm then install it from <a href="https://www.npmjs.com/get-npm">here</a>

3. cd ./TransAm

4. npm start
